Functions can make your code easier to read by replacing complicated lines of code with a meaningful name. They can also help to reduce repetition in your code. Take a look at lines 13 and 16 in sketch.js. The boolean expressions on these lines are very similar--both check if a particular coordinate has reached an edge of the canvas. Your task is to write a single function that can replace these boolean expressions.

Part 1

Define a function called isOutOfBounds() that takes three arguments: coordinate, minVal, and maxVal. coordinate will be the coordinate to check (x or y), minVal will be the lowest acceptable value of the coordinate. maxVal will be the highest acceptable value of the coordinate. The function should return true if the coordinate is out of bounds and false if the coordinate is in bounds.

Replace the boolean expressions on lines 13 and 16 with calls to isOutOfBounds(). The same function should work for both boolean expressions!

Part 2

Draw a rectangle in the centre of the canvas. The rectangle should be smaller than the the canvas (some background should be visible around the edge). Give it a fill colour other than white. The rectangle should appear under the ball (i.e. call the rect() function before calling circle() in draw()).

In draw(), add a conditional that calls isOutOfBounds() to check if the circle's coordinates are inside the rectangle. If either coordinate is out of bounds, set the circle's fill colour to match that of the rectangle. Otherwise, the circle's fill colour should be white.

When you are done, check your work using check-exercise2-1.html.
