Instructions

Open exercise2-2/sketch.js, add setup() and draw() and create a 600 x 600 canvas. What the video below that shows the expected output. Focus on how each circle moves in relation to the mouse. The yellow circle follows the mouse cursor but the other circles "mirror" the mouse movement on one or both axes. Mirror is not a technical term, it's just the word I'm using to describe the effect.

Add a function named mirrorCoordinate that takes one parameter called coordinate, representing a single coordinate (either x or y). The function will return a number, the mirrored version of the coordinate.

Fill in the function. You'll need to work out the maths yourself but here's a tip: if coordinate is N pixels away from one side of the canvas, its mirror is N pixels away from the opposite side. So, taking into account the size of the canvas, if coordinate is 20, your function should return 580. Your function should ideally only contain one, maybe two lines of code.

Now you're ready to complete your draw() function. Draw four circles and use the mirrorCoordinate() function to calculate the position of each circle. Feel free to choose different colours and/or make your circles a different size than the video shows.

When you are done, check your work using check-exercise2-2.html.

Expected output:

The maths for mirrorCoordinate() is very, very simple. Try not to overthink it!

Try to work out how / when to use mirrorCoordinate() to implement this sketch on your own but here's some more guidance if you get stuck:

  • One circle uses the raw, unmirrored location of the mouse.
  • One circle mirrors mouseX AND mouseY.
  • Two circles mirror only one of the mouse coordinates and use the raw mouse coordinate for the other axis.
